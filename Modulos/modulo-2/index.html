A proposta é oferecer um conjunto de módulos flexíveis, com conteúdos teóricos e práticos, que abrandem desde conceitos básicos de IA até aplicações mais avançadas em diversas áreas da programação.

Módulo 2: Desenvolvimento de Aplicações Básicas

Sub-módulo 1: Manipulação de Dados

- Lição 1: Entrada e Saída de Dados

- Pergunte à IA a seguinte frase: você pode me explicar sobre o conceito de programação?

- Pergunte o seguinte à IA: Crie um guia didático para o aprendizado de insersão e armazenamento de dados em linguagens de programação

Aqui nesta lição, você irá praticar o domínio sobre como armazenar dados em arquivos e no console, e há a possibilidade de especificar linguagens de programação para o entendimento específico de cada sintaxe.

-Tarefa:

Agora peça à IA que crie uma lista de exercícios para você praticar com a seguinte pergunta: você pode criar uma lista de tarefas sobre conceitos básicos da insersão e armazenamento de dados na linguagem (escolha a linguagem que deseja estudar)?

- Lição 2: Trabalhando com Strings

- Pergunte o seguinte à IA: Crie um guia didático para o aprendizado de strings e suas funções básicas em linguagens de programação

Aqui nesta lição, você irá praticar o domínio sobre a função string e suas funções básicas, aprendendo sobre sua fundamentação básica e também podendo especificar diferentes linguagens de programação.

-Tarefa:

Agora peça à IA que crie uma lista de exercícios para você praticar com a seguinte pergunta: você pode criar uma lista de tarefas sobre conceitos básicos de strings e suas funções básicas na linguagem (escolha a linguagem que deseja estudar)?

- Tarefa:

Agora peça à IA que crie uma lista de exercícios para você praticar, com a seguinte pergunta: você pode criar uma lista de tarefas sobre os operadores básicos na linguagem (escolha a linguagem que deseja estudar)?

- Lição 3: Manipulação de Arquivos

- Pergunte o seguinte à IA: Crie um guia didático para o aprendizado de leitura e escrita de diferentes formatos em linguagens de programação, formatos como texto, JSON, CSV, etc.

Aqui nesta lição, você irá praticar o domínio sobre a manipulação de dados em formatos de arquivos mais básicos como texto, CSV, JSON. Obviamente, para não limitarmos sua curiosidade, pesquise sobre outros formatos de arquivos além de diferentes linguagens de programação.

-Tarefa:

Agora peça à IA que crie uma lista de exercícios para você praticar com a seguinte pergunta: você pode criar uma lista de tarefas sobre conceitos básicos de leitura e escrita de diferentes formatos como texto, JSON, CSV, etc. na linguagem (escolha a linguagem que deseja estudar)?

Sub-módulo 2: Algoritmos Básicos

- Lição 1: Ordenação e Busca

- Pergunte o seguinte à IA: Crie um guia didático para o aprendizado de ordenação e busca e a sua aplicação em linguagens de programação

Aqui nesta lição, você irá praticar o domínio sobre ordenação e busca, aplicando os conceitos em qualquer linguagem de programação especificada.

-Tarefa:

Agora peça à IA que crie uma lista de exercícios para você praticar com a seguinte pergunta: você pode criar uma lista de tarefas sobre conceitos básicos de ordenação e busca na linguagem (escolha a linguagem que deseja estudar)?

- Lição 2: Recursão

- Pergunte o seguinte à IA: Crie um guia didático para o aprendizado dos princípios básicos de recursão e sua aplicação em linguagens de programação

Aqui nesta lição, você irá praticar o domínio sobre os princípios básicos ou também avançados da recursão em algoritmos, aprendendo sobre a sua aplicação em qualquer que seja a linguagem de programação especificada.

-Tarefa:

Agora peça à IA que crie uma lista de exercícios para você praticar com a seguinte pergunta: você pode criar uma lista de tarefas sobre conceitos básicos de recursão e sua aplicação na linguagem (escolha a linguagem que deseja estudar)?

- Lição 3: Estruturas de Dados Avançadas

- Pergunte o seguinte à IA: Crie um guia didático para o aprendizado de estruturas de dados como pilhas, filas e árvores simples em linguagens de programação

Aqui nesta lição, você irá praticar o domínio sobre estruturas de dados como pilhas, filas e árvores simples, além de claro, poder consultar a ferramenta acerca de diferentes tipos de estruturas de dados e de suas aplicações em diferentes linguagens de programação.

-Tarefa:

Agora peça à IA que crie uma lista de exercícios para você praticar com a seguinte pergunta: você pode criar uma lista de tarefas sobre conceitos básicos de estruturas de dados como pilhas, filas e árvores simples na linguagem (escolha a linguagem que deseja estudar)?

Lembre-se que mudanças podem ser feitas nas perguntas a fim de aumentar a complexidade das requisições para assim obter os melhores resultados possíveis, além de também poder modificá-las para tratar de outros assuntos específicos acerca do aprendizado de linguagens de programação.
